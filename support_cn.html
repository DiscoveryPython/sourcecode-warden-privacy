支持与帮助

欢迎使用 Source Code Warden！如果您在选择源文件、分配备份目录、选择命名方案或创建备份时遇到问题，可以先参考以下常见问题（FAQ）。

常见问题（FAQ）

Q: Source Code Warden 可以做什么？

A: Source Code Warden 可帮助您选择源文件或项目目录，为不同文件分配备份目标目录，选择备份命名方案，并创建安全、清晰、可追踪的备份副本。

Q: 为什么应用无法访问我的文件或备份目录？

A: 请确认您已授予文件或文件夹访问权限；如有需要，请重新选择源文件、项目目录或目标备份文件夹。

Q: 为什么 Normal Backup 在已有同名备份时会失败？

A: Normal Backup 使用严格的按日期命名规则。如果目标目录中已经存在同名备份，您可以选择其他目标目录、移除已有备份，或使用 Debug Version Backup 自动生成下一个可用版本后缀。

Q: 什么是 Debug Version Backup？

A: Debug Version Backup 会扫描目标目录并自动生成下一个可用后缀，例如 A、B、C 或 AA，从而避免覆盖已有备份文件。
