Source Code Warden 隐私政策

更新日期：2026年4月

1. 信息收集

Source Code Warden 不会向开发者收集、存储或传输用户的个人信息。

2. 本地文件与文件夹访问

Source Code Warden 可能会请求您选择本地文件、项目文件夹以及备份目标文件夹，以便您在设备上创建安全、清晰、可追踪的备份。您选择的文件或文件夹仅用于设备本地备份操作，不会上传给开发者。

3. 备份处理

Source Code Warden 的设计目标是在您的 Mac 本地处理所选源文件和项目目录。备份清单、目标文件夹分配、命名方案、生成的备份文件名、进度信息以及 Activity Log 均在设备本地处理。

如果应用在备份过程中创建临时工作数据，这些数据也仅用于完成您当前请求的备份操作，并保留在您的设备上。

4. 网络访问

Source Code Warden 的核心备份功能不依赖开发者运营的服务器。备份创建、目录分配、命名和日志记录均在您的 Mac 本地完成。

若未来某些功能涉及打开用户提供的在线资源或外部链接，您的设备将直接连接到相应第三方服务，而不会通过开发者服务器转发。

第三方服务在标准网络通信过程中可能会获得您的 IP 地址等技术信息，其信息处理方式受其自身隐私政策约束。

5. 第三方 SDK

Source Code Warden 不集成第三方统计分析或广告 SDK。

6. 数据安全

您的源文件、项目目录、备份文件、目标目录路径以及 Activity Log 均保留在您的设备上，除非您自行选择保存、移动、复制或分享它们。Source Code Warden 不会将您的源代码或备份数据上传给开发者。
